1-Colocar a pasta TitanSoftware em C:\xampp\htdocs
2-Abrir o  xampp e dar start no Apache e no MySQL
3-Acessar o link http://localhost/TitanSoftware/insereproduto.php
4-Acessar o link http://localhost/phpmyadmin/
5-Copiar, colar e executar o comando SQL no phpMyAdmin, que está disponivel dentro da pasta TitanSoftware
6-Cadastrar o produto
